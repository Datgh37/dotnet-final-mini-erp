CHƯƠNG 1: TỔNG QUAN & KIẾN TRÚC
1. Phần 1: Tổng Quan & Kiến Trúc
1.1. Giới Thiệu Đề Tài
Ứng dụng: Mini ERP là một hệ thống quản lý tài nguyên doanh nghiệp (ERP) phiên bản tinh gọn cho cửa hàng nhỏ.
Hệ thống bao gồm: MiniERP_API (Web API backend xử lý logic nghiệp vụ); MiniERP_WebView (Frontend Web MVC cho giao diện người dùng)
Công nghệ cốt lõi:
• Backend (MiniERP_API)
o Runtime: .NET 10.0
o Framework: ASP.NET Core Web API
o Database: SQL Server
o Data Access: ADO.NET (Microsoft.Data.SqlClient)
o Mapping: AutoMapper (DTO mapping)
o API Documentation: NSwag (Swagger UI)
o Architecture Pattern: Repository & Service Pattern
• Frontend (MiniERP_WebView)
o Framework: ASP.NET Core MVC
o Runtime: .NET 10.0
o Styling: Bootstrap 5 + HTML5
Link Source Code: https://github.com/Datgh37/dotnet-final-mini-erp
Các nghiệp vụ chính:
STTNghiệp vụMô tả1Quản lý Người dùng & Phân quyềnQuản lý tài khoản người dùng, băm mật khẩu và phân quyền vai trò (Roles) cho nhân viên.2Quản lý sản phẩmCRUD thông tin sản phẩm, danh mục hàng hóa, thương hiệu và đơn vị tính.3Quản lý đối tácCRUD thông tin chi tiết khách hàng và nhà cung cấp.4Nhập – Xuất khoThực hiện phiếu nhập kho (khi mua), phiếu xuất kho (khi bán), lưu nhật ký biến động và theo dõi số lượng tồn kho thực tế.5Quản lý mua hàngTạo đơn đặt hàng từ nhà cung cấp (Purchase Order), theo dõi ngày dự kiến và ngày nhận hàng thực tế.6Quản lý bán hàngTạo đơn hàng bán (Sales Order), chi tiết đơn hàng, quản lý phương thức thanh toán và địa chỉ giao hàng.7Tính toán doanh thuTự động tính tổng tiền đơn hàng dựa trên số lượng, đơn giá và chiết khấu.8Thống kê báo cáoTổng hợp và báo cáo doanh thu theo các khoảng thời gian (ngày, tháng, năm).9Bảo toàn dữ liệuXóa mềm (Soft Delete) tất cả dữ liệu (không xóa vật lý)1.2. Kiến Trúc Hệ Thống (3-Layer Architecture)

Giải thích:
• Presentation Layer: Nhận yêu cầu từ người dùng thông qua API, kiểm tra dữ liệu đầu vào và trả về JSON response.
• Business Logic Layer: Chứa các quy trình nghiệp vụ cốt lõi, thực hiện tính toán và điều phối các hành động giữa tầng API và tầng dữ liệu.
• Data Access Layer: Thực hiện kết nối và tương tác trực tiếp với cơ sở dữ liệu bằng ADO.NET thông qua các mẫu Repository.
• Database Layer: Lưu trữ dữ liệu vật lý và xử lý các nghiệp vụ phức tạp trực tiếp bên trong SQL Server bằng Stored Procedures.

1.4. Luồng Xử Lý Request
Client (Postman/Frontend)
    │
    ▼
[HTTP Request] → Controller (Validate + Map DTO)
                      │
                      ▼
                 Service (Business Logic)
                      │
                      ▼
                 Repository (ADO.NET → SQL Server)
                      │
                      ▼
                 [SQL Query Execute]
                      │
                      ▼
                 Repository (Map DataReader → Model)
                      │
                      ▼
                 Service (Process + Return)
                      │
                      ▼
                 Controller (Map → Response DTO)
                      │
                      ▼
              [HTTP Response JSON]

2.1. Nguyên Tắc Soft Delete
Tất cả các bảng đều có cột IsDeleted (BIT, default 0).
Khi "xóa", chỉ UPDATE IsDeleted = 1, KHÔNG dùng DELETE.
Tất cả câu SELECT đều thêm WHERE IsDeleted = 0.

2.4. Stored Procedures Quan Trọng
2.4.1. Tạo đơn bán hàng & Trừ kho (sp_CreateSalesOrder)
• Chức năng: Dùng để xử lý trọn gói một đơn hàng bán ra.
• Các bước thực hiện:
o Tạo bản ghi mới trong bảng SalesOrders.
o Đọc danh sách sản phẩm từ định dạng XML và chèn vào bảng SalesOrderItems.
o Kiểm tra tồn kho từng sản phẩm; nếu thiếu sẽ báo lỗi và hủy toàn bộ giao dịch.
o Tự động trừ số lượng trong bảng Products.
o Ghi nhật ký xuất kho vào bảng StockMovements.
o Tự động tính toán lại tổng tiền của đơn hàng.
2.4.2. Xác nhận nhập hàng (sp_ReceivePurchaseOrder)
• Chức năng: Dùng khi hàng từ nhà cung cấp về tới kho.
• Các bước thực hiện:
o Cập nhật trạng thái đơn mua hàng (PurchaseOrders) thành 'RECEIVED'.
o Cập nhật ngày nhận hàng và người nhận.
o Tự động cộng thêm số lượng tồn kho vào bảng Products.
o Ghi nhật ký nhập kho vào bảng StockMovements.
2.4.3. Báo cáo doanh thu (sp_GetRevenueReport)
• Chức năng: Kết xuất dữ liệu thống kê kinh doanh trong một khoảng thời gian.
• Các bước thực hiện:
o Lọc các đơn hàng có trạng thái 'COMPLETED' và không bị xóa.
o Nhóm dữ liệu theo ngày.
o Tính tổng số lượng đơn hàng và tổng doanh thu của từng ngày trong khoảng từ ngày FromDate đến ToDate.
2.4.4. Xóa mềm tổng quá (sp_SoftDelete)
• Chức năng: Một công cụ dùng chung để xóa dữ liệu an toàn cho tất cả các bảng.
• Các bước thực hiện:
o Nhận tên bảng và ID cần xóa làm tham số.
o Thay vì xóa vĩnh viễn, SP này cập nhật cột IsDeleted = 1 và cập nhật thời gian UpdatedAt.
o Giúp bảo toàn dữ liệu để phục vụ việc đối soát và báo cáo lịch sử.

3.7.2. HTTP Status Codes
Status CodeÝ nghĩaKhi nào dùng200 OKThành côngGET, PUT, DELETE thành công201 CreatedTạo thành côngPOST tạo mới resource400 Bad RequestLỗi nghiệp vụValidation fail, tồn kho không đủ404 Not FoundKhông tìm thấyResource không tồn tại hoặc đã bị soft delete500 Internal Server ErrorLỗi serverException không xử lý được3.7.3. Soft Delete Convention
• Tất cả DELETE API thực hiện Soft Delete (UPDATE IsDeleted = 1)
• Tất cả GET API tự động filter WHERE IsDeleted = 0
• Dữ liệu đã soft delete không xuất hiện trong kết quả trả về

2.2. Product Repository (ADO.NET Implementation)
File: MiniERP.DAL/Repositories/ProductRepository.cs
⚠️ QUAN TRỌNG:
Đây là ví dụ mẫu hoàn chỉnh cho ADO.NET Repository.
Các repository khác (Customer, Order...) làm tương tự pattern này.
using Microsoft.Data.SqlClient;
using MiniERP.DAL.DbContext;
using MiniERP.DAL.Interfaces;
using MiniERP.Models;
using System.Data;

namespace MiniERP.DAL.Repositories;

public class ProductRepository : IProductRepository
{
    private readonly SqlConnectionFactory _factory;

    public ProductRepository(SqlConnectionFactory factory)
    {
        _factory = factory;
    }

    // ========== GET ALL (có filter Soft Delete) ==========
    public async Task<IEnumerable<Product>> GetAllAsync()
    {
        var products = new List<Product>();
        const string sql = @"
            SELECT p.*, c.CategoryName 
            FROM Products p
            INNER JOIN Categories c ON p.CategoryId = c.CategoryId
            WHERE p.IsDeleted = 0
            ORDER BY p.CreatedAt DESC";

        using var connection = _factory.CreateConnection();
        using var command = new SqlCommand(sql, connection);
        
        await connection.OpenAsync();
        using var reader = await command.ExecuteReaderAsync();
        
        while (await reader.ReadAsync())
        {
            products.Add(MapToProduct(reader));
        }
        
        return products;
    }

    // ========== GET BY ID ==========
    public async Task<Product?> GetByIdAsync(int id)
    {
        const string sql = @"
            SELECT p.*, c.CategoryName 
            FROM Products p
            INNER JOIN Categories c ON p.CategoryId = c.CategoryId
            WHERE p.ProductId = @ProductId AND p.IsDeleted = 0";

        using var connection = _factory.CreateConnection();
        using var command = new SqlCommand(sql, connection);
        command.Parameters.AddWithValue("@ProductId", id);
        
        await connection.OpenAsync();
        using var reader = await command.ExecuteReaderAsync();
        
        if (await reader.ReadAsync())
            return MapToProduct(reader);
        
        return null;
    }

    // ========== CREATE ==========
    public async Task<int> CreateAsync(Product product)
    {
        const string sql = @"
            INSERT INTO Products 
                (ProductCode, ProductName, CategoryId, Unit, Price, CostPrice, StockQuantity)
            VALUES 
                (@ProductCode, @ProductName, @CategoryId, @Unit, @Price, @CostPrice, @StockQuantity);
            SELECT SCOPE_IDENTITY();";

        using var connection = _factory.CreateConnection();
        using var command = new SqlCommand(sql, connection);
        
        command.Parameters.AddWithValue("@ProductCode", product.ProductCode);
        command.Parameters.AddWithValue("@ProductName", product.ProductName);
        command.Parameters.AddWithValue("@CategoryId", product.CategoryId);
        command.Parameters.AddWithValue("@Unit", product.Unit);
        command.Parameters.AddWithValue("@Price", product.Price);
        command.Parameters.AddWithValue("@CostPrice", product.CostPrice);
        command.Parameters.AddWithValue("@StockQuantity", product.StockQuantity);
        
        await connection.OpenAsync();
        var result = await command.ExecuteScalarAsync();
        return Convert.ToInt32(result);
    }

    // ========== UPDATE ==========
    public async Task<bool> UpdateAsync(Product product)
    {
        const string sql = @"
            UPDATE Products SET
                ProductName = @ProductName,
                CategoryId = @CategoryId,
                Unit = @Unit,
                Price = @Price,
                CostPrice = @CostPrice,
                UpdatedAt = GETDATE()
            WHERE ProductId = @ProductId AND IsDeleted = 0";

        using var connection = _factory.CreateConnection();
        using var command = new SqlCommand(sql, connection);
        
        command.Parameters.AddWithValue("@ProductId", product.ProductId);
        command.Parameters.AddWithValue("@ProductName", product.ProductName);
        command.Parameters.AddWithValue("@CategoryId", product.CategoryId);
        command.Parameters.AddWithValue("@Unit", product.Unit);
        command.Parameters.AddWithValue("@Price", product.Price);
        command.Parameters.AddWithValue("@CostPrice", product.CostPrice);
        
        await connection.OpenAsync();
        var rows = await command.ExecuteNonQueryAsync();
        return rows > 0;
    }

    // ========== SOFT DELETE (Xóa mềm) ==========
    public async Task<bool> SoftDeleteAsync(int id)
    {
        const string sql = @"
            UPDATE Products SET 
                IsDeleted = 1, 
                UpdatedAt = GETDATE()
            WHERE ProductId = @ProductId AND IsDeleted = 0";

        using var connection = _factory.CreateConnection();
        using var command = new SqlCommand(sql, connection);
        command.Parameters.AddWithValue("@ProductId", id);
        
        await connection.OpenAsync();
        var rows = await command.ExecuteNonQueryAsync();
        return rows > 0;
    }

    // ========== HELPER: Map DataReader → Product ==========
    private static Product MapToProduct(SqlDataReader reader)
    {
        return new Product
        {
            ProductId = reader.GetInt32("ProductId"),
            ProductCode = reader.GetString("ProductCode"),
            ProductName = reader.GetString("ProductName"),
            CategoryId = reader.GetInt32("CategoryId"),
            Unit = reader.GetString("Unit"),
            Price = reader.GetDecimal("Price"),
            CostPrice = reader.GetDecimal("CostPrice"),
            StockQuantity = reader.GetInt32("StockQuantity"),
            IsDeleted = reader.GetBoolean("IsDeleted"),
            CreatedAt = reader.GetDateTime("CreatedAt"),
            UpdatedAt = reader.IsDBNull("UpdatedAt") 
                ? null 
                : reader.GetDateTime("UpdatedAt"),
            CategoryName = reader.IsDBNull("CategoryName") 
                ? null 
                : reader.GetString("CategoryName")
        };
    }
}
2.3. Inventory Repository (Ví dụ Nhập/Xuất Kho)
File: MiniERP.DAL/Repositories/InventoryRepository.cs
using Microsoft.Data.SqlClient;
using MiniERP.DAL.DbContext;
using MiniERP.DAL.Interfaces;
using MiniERP.Models;

namespace MiniERP.DAL.Repositories;

public class InventoryRepository : IInventoryRepository
{
    private readonly SqlConnectionFactory _factory;

    public InventoryRepository(SqlConnectionFactory factory)
    {
        _factory = factory;
    }

    /// <summary>
    /// Tạo phiếu nhập/xuất kho + cập nhật tồn kho (Transaction)
    /// </summary>
    public async Task<int> CreateTransactionAsync(InventoryTransaction transaction)
    {
        using var connection = _factory.CreateConnection();
        await connection.OpenAsync();
        
        // ★ Dùng Transaction để đảm bảo tính toàn vẹn dữ liệu
        using var dbTransaction = connection.BeginTransaction();
        
        try
        {
            // 1. Nếu xuất kho → kiểm tra tồn kho
            if (transaction.TransactionType == "Export")
            {
                const string checkSql = @"
                    SELECT StockQuantity FROM Products 
                    WHERE ProductId = @ProductId AND IsDeleted = 0";
                    
                using var checkCmd = new SqlCommand(checkSql, connection, dbTransaction);
                checkCmd.Parameters.AddWithValue("@ProductId", transaction.ProductId);
                
                var stock = (int)(await checkCmd.ExecuteScalarAsync() ?? 0);
                if (stock < transaction.Quantity)
                    throw new Exception($"Tồn kho không đủ! Hiện có: {stock}");
            }

            // 2. Insert InventoryTransaction
            const string insertSql = @"
                INSERT INTO InventoryTransactions 
                    (TransactionCode, ProductId, TransactionType, 
                     Quantity, UnitPrice, TransactionDate, Note)
                VALUES 
                    (@Code, @ProductId, @Type, @Qty, @Price, @Date, @Note);
                SELECT SCOPE_IDENTITY();";

            using var insertCmd = new SqlCommand(insertSql, connection, dbTransaction);
            insertCmd.Parameters.AddWithValue("@Code", transaction.TransactionCode);
            insertCmd.Parameters.AddWithValue("@ProductId", transaction.ProductId);
            insertCmd.Parameters.AddWithValue("@Type", transaction.TransactionType);
            insertCmd.Parameters.AddWithValue("@Qty", transaction.Quantity);
            insertCmd.Parameters.AddWithValue("@Price", transaction.UnitPrice);
            insertCmd.Parameters.AddWithValue("@Date", transaction.TransactionDate);
            insertCmd.Parameters.AddWithValue("@Note", 
                (object?)transaction.Note ?? DBNull.Value);

            var newId = Convert.ToInt32(await insertCmd.ExecuteScalarAsync());

            // 3. Cập nhật tồn kho Products
            var updateOp = transaction.TransactionType == "Import" ? "+" : "-";
            var updateSql = $@"
                UPDATE Products SET 
                    StockQuantity = StockQuantity {updateOp} @Qty,
                    UpdatedAt = GETDATE()
                WHERE ProductId = @ProductId AND IsDeleted = 0";

            using var updateCmd = new SqlCommand(updateSql, connection, dbTransaction);
            updateCmd.Parameters.AddWithValue("@Qty", transaction.Quantity);
            updateCmd.Parameters.AddWithValue("@ProductId", transaction.ProductId);
            await updateCmd.ExecuteNonQueryAsync();

            // 4. Commit transaction
            await dbTransaction.CommitAsync();
            return newId;
        }
        catch
        {
            await dbTransaction.RollbackAsync();
            throw;
        }
    }
}

    }

    /// <summary>
    /// Thống kê doanh thu theo tháng trong năm
    /// </summary>
    public async Task<IEnumerable<MonthlyRevenue>> GetMonthlyRevenueAsync(int year)
    {
        var results = new List<MonthlyRevenue>();
        
        const string sql = @"
            SELECT 
                MONTH(o.OrderDate) AS [Month],
                COUNT(o.OrderId) AS TotalOrders,
                SUM(o.TotalAmount) AS TotalRevenue
            FROM Orders o
            WHERE o.IsDeleted = 0 
              AND o.Status = N'Completed'
              AND YEAR(o.OrderDate) = @Year
            GROUP BY MONTH(o.OrderDate)
            ORDER BY [Month]";

        using var connection = _factory.CreateConnection();
        using var command = new SqlCommand(sql, connection);
        command.Parameters.AddWithValue("@Year", year);

        await connection.OpenAsync();
        using var reader = await command.ExecuteReaderAsync();

        while (await reader.ReadAsync())
        {
            results.Add(new MonthlyRevenue
            {
                Month = reader.GetInt32(0),
                TotalOrders = reader.GetInt32(1),
                TotalRevenue = reader.GetDecimal(2)
            });
        }

        return results;
    }
}

using MiniERP.BLL.Interfaces;
using MiniERP.DAL.Interfaces;
using MiniERP.Models;

namespace MiniERP.BLL.Services;

public class OrderService : IOrderService
{
    private readonly IOrderRepository _orderRepo;
    private readonly IProductRepository _productRepo;

    public OrderService(
        IOrderRepository orderRepo, 
        IProductRepository productRepo)
    {
        _orderRepo = orderRepo;
        _productRepo = productRepo;
    }

    /// <summary>
    /// Tạo đơn hàng mới:
    /// 1. Validate sản phẩm tồn tại
    /// 2. Kiểm tra tồn kho
    /// 3. Tính SubTotal cho từng dòng
    /// 4. Tính TotalAmount cho đơn hàng
    /// 5. Lưu vào DB
    /// </summary>
    public async Task<int> CreateOrderAsync(Order order)
    {
        if (order.OrderDetails == null || order.OrderDetails.Count == 0)
            throw new ArgumentException("Đơn hàng phải có ít nhất 1 sản phẩm!");

        // Sinh mã đơn hàng tự động
        order.OrderCode = $"DH-{DateTime.Now:yyyyMMddHHmmss}";
        order.OrderDate = DateTime.Now;
        order.Status = "Pending";

        decimal totalAmount = 0;

        foreach (var detail in order.OrderDetails)
        {
            // Kiểm tra sản phẩm
            var product = await _productRepo.GetByIdAsync(detail.ProductId);
            if (product == null)
                throw new KeyNotFoundException(
                    $"Sản phẩm ID = {detail.ProductId} không tồn tại!");

            // Kiểm tra tồn kho
            if (product.StockQuantity < detail.Quantity)
                throw new Exception(
                    $"Sản phẩm '{product.ProductName}' không đủ tồn kho! " +
                    $"Hiện có: {product.StockQuantity}, Yêu cầu: {detail.Quantity}");

            // Lấy giá bán hiện tại
            detail.UnitPrice = product.Price;
            detail.SubTotal = detail.Quantity * detail.UnitPrice;
            totalAmount += detail.SubTotal;
        }

        order.TotalAmount = totalAmount;
        return await _orderRepo.CreateAsync(order);
    }

    public async Task<IEnumerable<Order>> GetAllOrdersAsync()
    {
        return await _orderRepo.GetAllAsync();
    }

    public async Task<Order?> GetOrderByIdAsync(int id)
    {
        return await _orderRepo.GetByIdAsync(id);
    }

    public async Task<bool> DeleteOrderAsync(int id)
    {
        return await _orderRepo.SoftDeleteAsync(id);
    }
}

using Microsoft.AspNetCore.Mvc;
using MiniERP.API.DTOs.Request;
using MiniERP.API.DTOs.Response;
using MiniERP.BLL.Interfaces;
using MiniERP.Models;

namespace MiniERP.API.Controllers;

[ApiController]
[Route("api/[controller]")]
public class ProductsController : ControllerBase
{
    private readonly IProductService _productService;

    public ProductsController(IProductService productService)
    {
        _productService = productService;
    }

    // GET: api/products
    [HttpGet]
    public async Task<IActionResult> GetAll()
    {
        var products = await _productService.GetAllProductsAsync();
        return Ok(ApiResponse<IEnumerable<Product>>
            .Ok(products, "Lấy danh sách sản phẩm thành công"));
    }

    // GET: api/products/5
    [HttpGet("{id}")]
    public async Task<IActionResult> GetById(int id)
    {
        var product = await _productService.GetProductByIdAsync(id);
        if (product == null)
            return NotFound(ApiResponse<object>
                .Fail($"Không tìm thấy sản phẩm ID = {id}"));

        return Ok(ApiResponse<Product>.Ok(product));
    }

    // POST: api/products
    [HttpPost]
    public async Task<IActionResult> Create(
        [FromBody] CreateProductRequest request)
    {
        var product = new Product
        {
            ProductCode = request.ProductCode,
            ProductName = request.ProductName,
            CategoryId = request.CategoryId,
            Unit = request.Unit,
            Price = request.Price,
            CostPrice = request.CostPrice,
            StockQuantity = request.StockQuantity
        };

        var newId = await _productService.CreateProductAsync(product);
        return CreatedAtAction(
            nameof(GetById), 
            new { id = newId },
            ApiResponse<object>.Ok(
                new { ProductId = newId }, 
                "Tạo sản phẩm thành công"));
    }

    // PUT: api/products/5
    [HttpPut("{id}")]
    public async Task<IActionResult> Update(
        int id, [FromBody] UpdateProductRequest request)
    {
        var product = new Product
        {
            ProductId = id,
            ProductName = request.ProductName,
            CategoryId = request.CategoryId,
            Unit = request.Unit,
            Price = request.Price,
            CostPrice = request.CostPrice
        };

        var result = await _productService.UpdateProductAsync(product);
        if (!result)
            return NotFound(ApiResponse<object>.Fail("Cập nhật thất bại"));

        return Ok(ApiResponse<object>.Ok(null!, "Cập nhật thành công"));
    }

    // DELETE: api/products/5 (Soft Delete)
    [HttpDelete("{id}")]
    public async Task<IActionResult> Delete(int id)
    {
        var result = await _productService.DeleteProductAsync(id);
        if (!result)
            return NotFound(ApiResponse<object>.Fail("Không tìm thấy sản phẩm"));

        return Ok(ApiResponse<object>
            .Ok(null!, "Xóa sản phẩm thành công (soft delete)"));
    }
}
